Code Execution Engine API

Welcome to the Code Execution Engine API documentation. This API allows you to execute Python code in isolated containers with custom package dependencies.

Available Endpoints

  • GET / - This documentation page
  • POST /containers - Create a new container with specified packages
  • GET /containers - List all active containers
  • DELETE /containers/{container_id} - Delete a specific container
  • POST /execute - Execute Python code in a container
  • POST /jobs - Create a scheduled job
  • GET /jobs - List all scheduled jobs
  • GET /logs - Get execution logs

For detailed API documentation, please visit the Swagger UI or ReDoc.
